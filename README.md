# 中国地级市数据可视化发布说明

## 文件

- `index.html`：发布页面。
- `data/完整版地级市.geojson`：带 2004-2024 年核心年鉴指标的地级市面 GeoJSON。

## 数据来源

- 行政区划边界：天地图行政区域可视化市级边界；台湾市级缺失面由同源天地图县级边界补充。
- 经济人口数据：中国城市年鉴数据 2004-2024。
- 审图号：GS（2024）0650号。
- 页面不加载全球底图瓦片，仅绘制中国行政区划边界与数据面，避免出现周边国家底图信息。

## 指标

页面动态读取 GeoJSON `metadata.metrics`。当前发布数据包含 206 个变量，支持在页面顶部搜索并选择任意变量着色。

默认详情面板展示：

- 地区生产总值
- 人口数（由地区生产总值 / 人均GDP反推）
- 行政土地面积
- 地区生产总值增长率
- 规模以上工业企业数量

若当前年份缺失某项数据，详情面板显示最近可用年份的数值。

## 更新日期

本发布包生成于 2026-06-30。

## 部署

使用静态 Web 服务发布本目录即可。页面资源均使用相对路径引用本地数据文件。

Cloudflare Pages 推荐设置：

- Framework preset：None
- Build command：留空
- Build output directory：`/`
