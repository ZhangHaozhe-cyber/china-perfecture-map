# 中国地级市数据可视化发布说明

## 文件

- `index.html`：发布页面。
- `data/prefecture_boundaries.geojson`：地级市边界 GeoJSON，不包含大体量面板属性。
- `data/city_panel_compact.json`：2004-2024 年城市面板指标紧凑数据，页面按行政区划代码与边界关联。

## 数据来源

- 行政区划边界：天地图行政区域可视化市级边界。
- 台湾显示：发布版地图使用天地图市级数据中的台湾省整体面要素显示台湾岛，无年鉴数据时以灰色填充。
- 南海诸岛与边界线：发布版保留天地图市级 GeoJSON 中的线状境界线要素，并在前端以深色虚线绘制。
- 经济人口数据：中国城市年鉴数据 2004-2024。
- 审图号：GS（2024）0650号。
- 页面不加载全球底图瓦片，仅绘制中国行政区划边界与数据面，避免出现周边国家底图信息。

## 指标

页面动态读取 `city_panel_compact.json` 的 `metadata.metrics`。当前发布数据包含 206 个变量，支持在页面顶部搜索并选择任意变量着色。

默认详情面板展示：

- 地区生产总值
- 人口数（由地区生产总值 / 人均GDP反推）
- 行政土地面积
- 地区生产总值增长率
- 规模以上工业企业数量

若当前年份缺失某项数据，详情面板显示最近可用年份的数值。

## 更新日期

本发布包生成于 2026-06-30。

## 部署

使用静态 Web 服务发布本目录即可。页面资源均使用相对路径引用本地数据文件。

Cloudflare Pages 推荐设置：

- Framework preset：None
- Build command：留空或 `exit 0`
- Build output directory：`.`（仓库根目录）

注意：Cloudflare Pages 单个静态文件大小上限为 25 MiB。本发布版已将原完整 GeoJSON 拆分为边界文件和紧凑面板数据文件，单个文件均低于该限制。
